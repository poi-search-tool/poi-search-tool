# POI搜索与下载工具

## 项目简介
POI（兴趣点）搜索与下载工具是一个基于R Shiny框架开发的网页应用程序，集成高德地图开放平台API，提供高效的POI数据获取、可视化展示和多格式导出功能。

## 核心功能

### 智能数据搜索
- 支持多种POI类型搜索（酒店、餐厅、学校、商场等）
- 按城市区域精准筛选
- 自动分页获取完整数据集
- 实时搜索状态提示

### 数据可视化展示
- 交互式数据表格（支持排序、筛选、分页）
- 地理信息地图分布
- 多标签页切换查看模式
- 响应式网页设计

### 数据导出管理
- CSV格式数据导出
- UTF-8编码支持中文字符
- 自动按"关键词_城市"命名规范
- 完整字段数据保存

## 技术架构

### 前端技术
- **R Shiny** - 网页应用框架
- **DT** - 交互式数据表格
- **Leaflet** - 地图可视化

### 后端技术  
- **httr** - HTTP请求处理
- **jsonlite** - JSON数据解析
- **dplyr** - 数据整理操作

### 数据服务
- **高德地图Place API** - POI数据源
- **星图地球** - 地图底图服务

## 安装部署

### 环境要求
```r
# 必需R包
install.packages(c("shiny", "DT", "leaflet", "httr", "jsonlite", "dplyr"))
```

运行方法

```r
# 方法一：在RStudio中运行
# 打开app.R文件，点击"Run App"

# 方法二：命令行运行
library(shiny)
runApp("app.R")
```

配置说明

本项目已集成高德地图API服务，开箱即用。如需更换API密钥，请修改app.R文件中的相关配置。

使用指南

基本操作流程

1. 输入搜索关键词（如：酒店、学校）
2. 选择目标城市（如：北京市）
3. 点击"开始搜索"获取数据
4. 在表格或地图标签页查看结果
5. 点击"下载CSV文件"导出数据

注意事项

· 确保网络连接正常
· API密钥需有效且未超出调用限额
· 地图显示需要网络加载资源
· 建议单次搜索不超过50页数据
